## 说明介绍

一个基于 VuePress 搭建的博客文档。目前计划写 3 个系列：

- 重学 JS(📝 进行中)
- ES6 进阶(📝 进行中)
- Webpack(📝 进行中)
- Vue(计划中)
- React(计划中)

写作不易，感觉写的不错的可以给个 Star⭐。如果有疑问或者发现错误，可以提 issues。

## 如何使用

```bash
# 1. clone该项目
git clone git@github.com:liujie2019/VuePress-Blog.git

# 2. 安装依赖
yarn install

# 3. 本地运行
yarn run docs:dev
```
